# InfraWatch-
InfraWatch is a crowdsourced infrastructure reporting tool that allows users to report and track infrastructure issues like potholes, broken streetlights, faulty traffic signals, and other infrastructure issues. Built with HTML, CSS, JavaScript, Firebase Authentication, Firestore, and Cloudinary for efficient issue tracking and management.
